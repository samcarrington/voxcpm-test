# Product

## Register

product

## Users

Solo developer evaluating and comparing TTS model outputs. Primary workflow: enter text/voice description, generate audio, save outputs, upload reference voices for cloning. Single-user, localhost-only, used for model comparison and iteration.

## Product Purpose

Interactive web front-end for VoxCPM2 TTS model. Replaces CLI-only interaction with a browser form that supports text-to-speech generation, voice cloning via reference audio, and live streaming playback — all while comparing model parameters interactively.

## Brand Personality

Sharp, precise, unapologetic. Designed by a developer for a developer — looks like something you'd want to leave open on a second monitor while comparing model outputs for hours.

## Anti-references

Not a boring form. Not SaaS cream. Not a dashboard. It should feel like an instrument — focused, deliberate, with character.

## Design Principles

1. Instrument over form — every control exists for a reason, visually weighted by its importance
2. Feedback first — status, loading, and results should be immediately legible at a glance
3. Dark by default — long-form comparative work belongs in low ambient light

## Accessibility & Inclusion

WCAG AA contrast compliance. Full keyboard navigation. No motion required.
